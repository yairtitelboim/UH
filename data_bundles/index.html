Houston Map Visualization - Data Bundles

This page provides access to the GeoJSON data files required for the Houston Map Visualization project. Since these files are too large to be included in the GitHub repository, they are made available here for download.

Available Bundles

Option 1: Complete Dataset
Contains all GeoJSON files needed for the project in a single bundle.
Size: ~191 MB compressed
Download
Option 2: Large Files Only
Contains only the largest GeoJSON files:
  • houston_buildings.geojson (~276 MB uncompressed)
  • Pipelines.geojson (~417 MB uncompressed)
Size: ~129 MB compressed
Download
Option 3: Medium-sized Files
Contains medium-sized GeoJSON files including segments, boundaries, and surface water data.
Size: ~87 MB compressed
Download
Option 4: Small Files
Contains all smaller GeoJSON files (under 10 MB each), including points of interest, census blocks, and other smaller datasets.
Size: ~11 MB compressed
Download

Installation Instructions

After downloading, extract the zip file(s) to the remote/public directory of your Houston Map Visualization project.

You can also use the provided download_data.sh script in the project which will automatically download and extract these files for you.
